# IT Support Ticketing Lab

## Overview

This repository demonstrates an IT support ticketing workflow similar to what is used in enterprise help desks.

The purpose of this lab is to document common IT support incidents from initial user report through troubleshooting, resolution, and ticket closure.

---

## Skills Demonstrated

- Incident Documentation
- Troubleshooting Methodology
- Windows User Support
- Active Directory Account Support
- Password Reset Procedures
- User Communication
- Root Cause Analysis
- Technical Documentation
- Ticket Lifecycle Management
- Git & GitHub Documentation
