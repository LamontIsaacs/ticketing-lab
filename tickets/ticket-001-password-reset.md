# Ticket 001 – Password Reset / Account Lockout

## Ticket Information

- **Ticket ID:** INC-0001
- **Category:** Account Access
- **Priority:** Medium
- **Status:** Closed
- **Assigned To:** IT Support Technician

---

## User Issue

The user reported being unable to sign into their Windows account after multiple unsuccessful login attempts.

---

## Initial Assessment

The account was verified to be locked due to repeated failed login attempts. No indication of hardware or network issues.

---

## Troubleshooting Steps

1. Verified the user's identity.
2. Confirmed the account was locked.
3. Reset the user's password.
4. Unlocked the account.
5. Had the user sign in with the temporary password.
6. Confirmed successful login.

---

## Resolution

The password was reset successfully, the account was unlocked, and the user regained access.

---

## Root Cause

Multiple incorrect password attempts triggered the account lockout policy.

---

## Lessons Learned

Advised the user to use a password manager and contact IT before repeated failed login attempts result in another lockout.

---

## Ticket Closed

**Resolution Time:** 15 minutes

**Final Status:** Closed
