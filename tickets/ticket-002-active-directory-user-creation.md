# Ticket 002 – Active Directory User Creation

## Ticket Information

- **Ticket ID:** INC-0002
- **Category:** User Account Management
- **Priority:** Medium
- **Status:** Closed
- **Assigned To:** IT Support Technician

---

## User Request

A department manager requested a new user account for a recently hired employee who needed access to the Windows domain.

---

## Initial Assessment

Confirmed that the request involved creating a new Active Directory user account and preparing the account for first-time login.

---

## Troubleshooting / Action Steps

1. Verified the new employee information.
2. Opened Active Directory Users and Computers.
3. Navigated to the correct organizational unit.
4. Created a new user account.
5. Assigned a temporary password.
6. Selected the option requiring the user to change password at next logon.
7. Verified the account appeared in Active Directory.
8. Confirmed the account was ready for first-time sign-in.

---

## Resolution

The Active Directory user account was created successfully and prepared for initial login.

---

## Root Cause

New employee onboarding required creation of a domain user account.

---

## Lessons Learned

Accurate user information and proper documentation are important during onboarding to prevent account access delays.

---

## Ticket Closed

**Resolution Time:** 20 minutes

**Final Status:** Closed
